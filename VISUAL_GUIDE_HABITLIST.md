## Animation Effects

### Card Entrance Animation
```
Timeline: 0ms â†’ 200ms

0ms:    Card at opacity 0, slightly lower
100ms:  Card animating up, fading in
200ms:  Card fully visible at rest
```

### Button Interactions
```
HOVER:      Scale up 105%, increase shadow
CLICK:      Show spinner, disable button
COMPLETE:   Fade to green, show success message
```

### Message Appearance
```
Success:    Fade in, stay 3 seconds, fade out
Error:      Fade in, stay until dismissed
Loading:    Spinner rotates continuously
```
